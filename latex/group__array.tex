\doxysubsection{Detailed Description}
Creating a two dimensional array. 

\begin{DoxyAuthor}{Author}
Jesper Andersson 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
26/11-\/2020
\end{DoxyDate}
This is a program made for creating and managing two dimensional arrays with a struct representing the array and its data. With it you can edit and view values at specific points in the array.
